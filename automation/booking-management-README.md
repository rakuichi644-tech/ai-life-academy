# 予約管理システムの設定

無料説明会の日程を運営側で追加・削除し、予約者のステータスをGoogleスプレッドシートで管理する仕組みです。

## できること

- 予約枠の追加・削除
- 残席表示
- 残席0の満員御礼表示
- 予約者へのZoomリンク自動メール
- 説明会後に配布するクーポン対象者の管理
- スプレッドシート上で「未申込」「説明会参加済み」「決済済み」「返金済み」などを管理

## 設定手順

1. Google Apps Scriptを新規作成します。
2. `booking-management.gs` の中身を貼り付けます。
3. `setupBookingSystem` を1回実行し、スプレッドシートと管理キーを作成します。
4. ウェブアプリとしてデプロイし、実行ユーザーを自分、アクセス権を全員にします。
5. 発行されたURLを `booking-slots.js` の `apiEndpoint` に設定します。
6. `admin.html` を開き、Apps Script URLと管理キーを入力します。

## スプレッドシート

初回実行時に `AI LIFE ACADEMY_予約管理` が自動作成されます。

- `予約枠`: 日程、定員、残席、公開状態を管理
- `予約者管理`: 予約者情報、説明会ステータス、決済ステータス、返金ステータス、購入者ステータスを管理

## Stripeクーポン

Stripe側でクーポンコード `FS20260701` を作成し、22万円から55,000円引きになるよう設定してください。
Payment Linksで「プロモーションコードを許可」を有効にすると、購入者が決済ページでコードを入力できます。
